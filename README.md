# pico-8-zombie-game

Zombie survival game for Pico-8. 🧟‍♂️

## Getting started

To run this game on Pico-8, follow these steps
* Open Pico-8
* Move the game file (.p8) to the carts directory of Pico-8
  * On MacOS this is `~/Library/Application Support/pico-8/carts/`
* Run `load` command on the game file
  * E.g. `load zombiegame.p8`
* Now your game cart is loaded
* Run game by runnning `run` command
